# FastAPI Blogicum

Перенос Django проекта Blogicum (6 спринт) на FastAPI.

## Технологии

- **FastAPI** — современный веб-фреймворк
- **SQLAlchemy** — ORM для работы с БД
- **Pydantic** — валидация данных
- **SQLite** — база данных (как в Django проекте)
